Lodging System – Final Project
(Versión en español abajo)

Web application developed as a final integrative project.
It allows complete management of tourist lodgings, including bookings, payments, and user administration (owners and guests).
Built with a modern architecture using Django (backend) and Vue.js (frontend), running inside Docker for a consistent and professional development environment.

 Technologies Used
 Backend

Python 3.11+
Django 5
Django REST Framework
Docker & Docker Compose
SQLite / PostgreSQL
JWT Authentication
Serializers, ViewSets & Custom Permissions

 Frontend

Vue.js 3
Pinia (state management)
Vue Router
Axios (API consumption)
Native CSS / TailwindCSS

 Additional Tools

Visual Studio Code
 GitHub


 Main Features

User registration and login (owners and guests)
Create, edit and delete lodgings
Interactive booking calendar
Payment management
Owner dashboard

Sistema de Alojamientos – Proyecto Final


Aplicación web desarrollada como proyecto final integrador.
Permite la gestión completa de alojamientos turísticos, incluyendo reservas, pagos y administración de usuarios (propietarios y huéspedes).
Fue creada con una arquitectura moderna basada en Django (backend) y Vue.js (frontend), utilizando Docker para un entorno de desarrollo profesional y reproducible.

Tecnologías utilizadas
 Backend

Python 3.11+
Django 5
Django REST Framework
Docker y Docker Compose
SQLite / PostgreSQL
Autenticación JWT
Serializadores, ViewSets y permisos personalizados

 Frontend

Vue.js 
Pinia (manejo de estado)
Vue Router
Axios (consumo de API REST)
CSS 

 Herramientas adicionales

Visual Studio Code
 GitHub



 Funcionalidades principales

 Registro e inicio de sesión de usuarios (propietarios y huéspedes)
 Creación, edición y eliminación de alojamientos
 Calendario interactivo de reservas
 Gestión de pagos
 Panel de administración para propietarios
 API REST segura con tokens JWT
 Interfaz moderna, responsiva y fácil de usar
